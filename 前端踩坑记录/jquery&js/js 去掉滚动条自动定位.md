# js 去掉滚动条自动定位
```
history.scrollRestoration = 'manual' // 将滚动条的自动定位设为手动的
```

> history.scrollRestoration 
允许Web应用程序在历史导航上显式地设置默认滚动恢复行为。此属性可以是自动的（auto）或者手动的（manual）——MDN
